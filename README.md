# 用藥記錄 (Medication)

記錄一段用藥療程：設定總天數、要吃的時段（早 / 中 / 晚 / 睡前）與每個時段的劑量，之後每天打勾記錄有沒有吃

## 適合誰

- 拿到一週療程的抗生素，想確認每天有按時吃完
- 早晚各一顆的長期用藥，需要一個地方打勾不漏吃
- 想回頭看整段療程哪幾餐漏掉

## 兩個畫面

右上角 nav bar 的藥丸 icon 點開是每日面板，側邊欄的「用藥記錄」點開是整段療程的大表格，兩邊資料同步

### nav bar 面板（每天打勾）

1. 還沒設定療程時，面板直接是設定表單
2. 設定好之後，面板顯示今天該吃的每個時段、劑量與打勾框
3. icon 上的小數字是今天還沒吃的劑數，吃完就消失
4. 療程還沒開始或已經結束，面板會直接告訴你

### 側邊欄表格（看整段進度）

- 每一列是一天（標出星期，今天那列會 highlight）
- 每一欄是一個用藥時段，欄標題帶劑量
- 任何一格都能打勾，過去漏打的也能補
- 最上面有整體進度條，一眼看完成幾劑

## 設定項目

- 藥名：選填，只是顯示用
- 開始日：預設今天
- 用藥期間：總天數，至少 1 天
- 時段與劑量：勾選早 / 中 / 晚 / 睡前要吃的時段，各填各的劑量（自由文字，例如 1 顆、5ml）

改設定不會刪掉已經打過的紀錄，只是超出新範圍的不再顯示

## 權限

- `ui:statusBar`：在右上角 nav bar 放藥丸 icon
- `ui:sidebar`：在側邊欄放入口、開全療程表格
- `storage`：存療程設定與打勾紀錄
