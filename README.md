# 用藥記錄 (Medication)

記錄你正在吃的每一種藥：每種各設總天數、要吃的時段（早 / 中 / 晚 / 睡前）與每個時段的劑量，之後每天打勾記錄有沒有吃

## 適合誰

- 同時吃好幾種藥，想分開追蹤每種有沒有按時吃完
- 拿到一週療程的抗生素，想確認每天吃完
- 早晚各一顆的長期用藥，需要一個地方打勾不漏吃
- 想回頭看每種藥哪幾餐漏掉

## 兩個畫面

右上角 nav bar 的藥丸 icon 點開是每日面板，側邊欄的「用藥記錄」點開是管理畫面，兩邊資料同步

### nav bar 面板（每天打勾）

1. 列出所有藥，每種顯示今天該吃的時段、劑量與打勾框
2. icon 上的小數字是今天所有藥加起來還沒吃的劑數，吃完就消失
3. 點「＋ 新增」直接加一種藥，點「管理 / 看全部」開側邊欄畫面
4. 某種藥還沒開始或已經結束，會直接告訴你

### 側邊欄管理畫面（看進度、加減藥）

- 畫在內容區，左邊的 app 導覽列保留不動
- 每種藥一張卡：藥名、起訖日期、整體進度條，加上整段療程的打勾表格（每列一天，今天那列 highlight）
- 右上角「＋ 新增藥物」加新的藥，每張卡有「編輯」與「刪除」（刪除要按兩次確認）
- 任何一格都能打勾，過去漏打的也能補

## 設定每一種藥

- 藥名：選填，只是顯示用
- 開始日：預設今天
- 用藥期間：總天數，至少 1 天
- 時段與劑量：勾選早 / 中 / 晚 / 睡前要吃的時段，各填各的劑量（自由文字，例如 1 顆、5ml）
- 附註：自由填注意事項（例如 飯後吃、避免與某藥同服），會顯示在面板與管理卡片上

改設定不會刪掉已經打過的紀錄，只是超出新範圍的不再顯示

## 資料

存在外掛自己的 `data.json`，純本機，不會上傳：

```json
{
  "medications": [
    {
      "id": "med-xxxx",
      "name": "抗生素",
      "startDate": "2026-06-28",
      "days": 7,
      "slots": {
        "morning": { "enabled": true, "dose": "1 顆" },
        "noon": { "enabled": false, "dose": "" },
        "evening": { "enabled": true, "dose": "1 顆" },
        "bedtime": { "enabled": false, "dose": "" }
      }
    }
  ],
  "taken": {
    "med-xxxx": { "2026-06-28": { "morning": true, "evening": false } }
  }
}
```

舊版（單一療程）的資料會在載入時自動轉成這個格式，不會弄丟你已經打過的紀錄

## 語言

介面跟著 Hyday 的語言走，支援正體中文、簡體中文、English，切換 app 語言時畫面即時跟著換

## 權限

- `ui:statusBar`：在右上角 nav bar 放藥丸 icon
- `ui:sidebar`：在側邊欄放入口、開管理畫面
- `storage`：存藥物設定與打勾紀錄
